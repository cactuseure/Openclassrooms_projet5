
# Projet 5 - Openclassrooms

Le projet consiste à développer un blog professionnel.


## Deployment

**Step 1 :**

À la racine du projet, créez un `.env` et enregistré ces valeurs : 
- DB_HOST
- DB_NAME
- DB_USER
- DB_PASSWORD
- CONTACT_EMAIL

**Step 2 :**
```bash
  composer install
```

**Step 3 :**
```bash
  npm install
```

**Step 4 :**
```bash
  npm run scss
```
